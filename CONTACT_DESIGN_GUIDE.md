# 고합상사 Contact 페이지 - 시각 설계 가이드

## 📋 개요
Contact 페이지는 고객과의 직접적인 상호작용을 담당하는 페이지로, 신뢰감 있고 전문적이면서도 접근성 좋은 디자인을 추구합니다.

---

## 🎨 디자인 철학

### 핵심 가치
- **신뢰감**: 명확한 정보 구조와 일관된 디자인
- **전문성**: 세련된 타이포그래피와 정제된 컬러 팔레트
- **접근성**: 모든 사용자가 쉽게 이용할 수 있는 인터페이스
- **효율성**: 필요한 정보를 빠르게 찾을 수 있는 구조

---

## 🎯 페이지 구조

### 1. Contact Hero 섹션 (연락처 소개)
**목적**: 페이지 진입점, 명확한 CTA 제공

```
┌─────────────────────────────────┐
│    문의하기                      │
│  언제든지 편하게 연락주세요      │
└─────────────────────────────────┘
```

**디자인 특징**:
- 배경: 밝은 블루 그래디언트 (#fbfdff → #eef3ff)
- 텍스트 정렬: 중앙
- 제목: 32px, 브랜드 블루, 볼드(700)
- 부제목: 16px, 뮤트 색상

---

### 2. Contact Cards (연락처 정보)
**목적**: 다양한 연락 방법을 시각적으로 제시

#### 카드 레이아웃
```
┌─────────────┐
│      📞     │  (56x56px 원형 아이콘)
│  전화 문의  │  (16px, 볼드)
│ 063-471-17  │  (16px, 헤더 색상, 링크)
│ 평일 9-18시 │  (13px, 뮤트 색상)
└─────────────┘
```

**그리드 구성**:
- 데스크톱 (900px+): 4열
- 태블릿 (600-900px): 2열
- 모바일 (<600px): 1열
- 갭: 16px

**카드 스타일**:
- 배경: 흰색 (#ffffff)
- 패딩: 24px (모바일: 18px)
- 둥근 모서리: 12px
- 그림자: 기본 카드 그림자
- Hover 효과: -4px 상승, 그림자 강화

**아이콘**:
- Font Awesome 6.5.2 사용
- 크기: 24px
- 배경: 연한 블루 원형 (#f2f2ff)
- 색상: 브랜드 블루

#### 4가지 카드 타입
1. **전화 문의**
   - 아이콘: fas fa-phone
   - 정보: 전화번호 (tel: 링크)
   - 세부: 업무 시간

2. **이메일 문의**
   - 아이콘: fas fa-envelope
   - 정보: 이메일 (mailto: 링크)
   - 세부: "가장 빠른 응답"

3. **업무 시간**
   - 아이콘: fas fa-clock
   - 정보: 평일 9:00-18:00
   - 세부: "점심시간 제외"

4. **위치**
   - 아이콘: fas fa-map-marker-alt
   - 정보: 군산시 개정면
   - 세부: 오시는 길 링크

---

### 3. Board Section (문의 게시판)
**목적**: 더 구체적인 문의나 제안을 받는 공간

#### 섹션 헤더
```
┌──────────────────────────────┐
│  문의 게시판                 │
│  더 자세한 사항을 남겨주세요 │
└──────────────────────────────┘
```

**스타일**:
- 배경: 연한 그래디언트
- 제목: 28px, 브랜드 블루, 중앙 정렬
- 부제목: 15px, 뮤트 색상

#### 2열 레이아웃
```
┌─────────────────┬─────────────────┐
│ 문의 폼         │ 최근 문의 목록   │
│ (왼쪽)          │ (오른쪽)         │
└─────────────────┴─────────────────┘
```

- 데스크톱: 1:1 비율, 32px 갭
- 태블릿/모바일: 단일 열

## 📝 폼 디자인

### 입력 필드
```
┌─ 이름 (선택) ──────────────────────────┐
│ [이름을 입력하세요]                     │
└────────────────────────────────────────┘
```

**라벨 스타일**:
- 폰트: 14px, 600 무게
- 색상: #172033
- 마진: 0 0 6px 0

**입력 필드 스타일**:
- 패딩: 10px 14px
- 테두리: 1px solid #ddd
- 둥근 모서리: 8px
- 배경: #fff
- 포커스: 브랜드 블루 테두리 + 연한 블루 그림자

**플레이스홀더**:
- 색상: #aaa
- 폰트 크기: 14px (부모와 동일)

### 텍스트 영역
```
┌─ 문의 내용 * ──────────────────────────┐
│ [문의 내용을 입력해주세요              │
│                                        │
│                                        │
│                                        │
│                                        │
└────────────────────────────────────────┘
```

**특징**:
- 최소 높이: 5줄 (rows="5")
- 세로 크기 조정 가능 (resize: vertical)
- 줄 높이: 1.5

### 체크박스
```
☑ 비밀글 (작성자만 확인 가능)
```

**스타일**:
- 크기: 18px
- 액센트 컬러: 브랜드 블루
- 텍스트 색상: #172033
- 폰트 크기: 14px
- 커서: pointer

### 제출 버튼
```
┌──────────────────────────────────────┐
│           문의 등록                  │
└──────────────────────────────────────┘
```

**기본 상태**:
- 배경: 브랜드 블루
- 텍스트: 흰색
- 패딩: 11px 22px
- 폭: 100%
- 폰트: 14px, 600 무게

**Hover 상태**:
- 배경: #0f1790 (더 어두운 블루)
- 상승: -2px
- 그림자: 강화됨

**Active 상태**:
- 상승: 없음
- 그림자: 약화됨

- 풀 너비 버튼

## ♿ 접근성

### WCAG 2.1 AA 준수
- 색상 대비: 4.5:1 (정상 텍스트), 3:1 (큰 텍스트)
- 포커스 표시: 3px 아웃라인, 2px 오프셋
- 링크: 밑줄 또는 다른 색상으로 구분
- 라벨: 모든 입력 필드에 명확한 라벨

### 상호작용 요소
- 최소 터치 크기: 44x44px
- 충분한 여백: 요소 간 8-12px
- 클릭 가능 영역 명확: 링크는 색상과 밑줄로 표시
