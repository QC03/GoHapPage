# 고합상사 홈페이지 (GoHapPage)

## 개요
고합상사(청소용품 공급)를 소개하는 웹사이트 기획안 기반의 간단한 홈페이지 프로젝트입니다. 주요 목적은 회사 소개, 연혁(신뢰성 강조), 제품/서비스(청소도구 판매·카페트 임대·화장지 판매), 연락/지도 안내 제공입니다.

원본 기획안: [기획안.txt](기획안.txt)

## 대상
- 목표 대상: 청소용품에 관심 있는 사업자
- 예측 연령층: 30대~70대

## 주요 페이지 및 기능
- 메인: 회사 소개, 핵심 서비스(판매·임대) 소개, 시각적 이미지 배치
- 연혁: 회사 성장·활동을 연도별로 표시(어두운 배경에 중심 카드)
- 컨택: 전화·이메일 정보 및 게시판(문의용)
- 지도: 오시는 길(네이버 지도 임베드 + 교통 안내)
- 공통: 상단 네비게이션, 하단 푸터(사업자번호·영업시간·저작권)

## 디자인 요약
- 전체 톤: 심플하고 깔끔한 레이아웃
- 주 색상: 블루 (#1212a1) 기반의 밝은 느낌
- 네비게이션: 기본 height 18px, hover 시 24px로 확대, 폰트 unhover 12px hover 18px
- 마진/간격: 각 섹션별 20px 여백

## 페이지 구조(상위 마일스톤, 일별 단위)
- Day 1: 프로젝트 초기 세팅 및 레포 구조 생성
- Day 2: 메인 레이아웃 및 헤더/푸터 구현
- Day 3: 메인 콘텐츠(회사 소개) 섹션 구현
- Day 4: 제품/서비스 섹션(판매·임대) 구현
- Day 5: 연혁 섹션 디자인 및 콘텐츠 배치
- Day 6: 컨택(문의처 + 게시판 목업) 구현
- Day 7: 지도 임베드 및 교통 안내 작성
- Day 8: 반응형/접근성 점검 및 스타일 정리
- Day 9: 테스트(링크, 폼 제출, 지도 표시) 및 수정
- Day10: 배포 준비 및 간단한 배포 지침 작성

> 필요 시 Day 단위는 세부화 가능(요청 시 태스크 수준으로 확장).

## 기술 스택(추천 / 미확정)
- 프론트엔드: 정적 사이트(HTML/CSS/JS) 또는 React/Vue 중 선택
- 백엔드(문의 게시판 등): 필요 시 간단한 Node.js(Express) 또는 Django/Flask API
- 지도: 네이버 지도 임베드

## 배포(간단 권장)
- 정적 사이트: GitHub Pages, Netlify, Vercel 추천
- 동적 기능(게시판 등) 필요 시: Heroku, Render, 또는 간단한 VPS
