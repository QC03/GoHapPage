# 고합상사 홈페이지

## 소개
고합상사는 청소용품 판매·카페트 임대·화장지 판매 등을 제공하는 기업입니다. 이 프로젝트는 기획안(`기획안.txt`)을 바탕으로 회사 소개, 연혁, 제품/서비스, 문의 및 오시는 길을 안내하는 공식 홈페이지의 기본 템플릿을 제공합니다.

## 대상
- 목표 대상: 청소용품에 관심 있는 사업자
- 예상 연령층: 30대–70대

## 주요 페이지 및 기능
- 메인(About): 한눈에 회사 정보를 보여주는 소개 섹션
- 제품/서비스: 청소도구 판매, 카페트 임대, 화장지 판매 소개 섹션
- 연혁(History): 회사 연혁과 활동을 연도별로 표기
- 컨택(Contact): 전화번호·이메일 기반 연락 안내
- 지도(Map): 카카오맵(다음 RoughMap) 임베드 및 교통 안내
- 공통 요소: 상단 네비게이션, 하단 푸터(사업자등록번호·영업시간·카피라이트)

## 디자인 가이드라인
- 전체 톤: 심플하고 깔끔한 레이아웃
- 브랜드 컬러: 블루 #1212a1 (밝은 톤과 조합)
- 여백: 섹션 간 기본 20px
- 네비게이션: 기본 높이 18px, hover 시 24px로 확대(transition 적용)
- 폰트: 깔끔한 고딕 계열(웹 안전 폰트 또는 구글 폰트 사용 권장)

## 기술 스택(초안)
- 프론트엔드: 정적 HTML/CSS/JS (초기 템플릿)
- 백엔드: 문의 게시판 등 동적 기능 필요 시 별도 구현(예: Node.js/Express 또는 Flask)
- 지도: 카카오맵 RoughMap 임베드

## 로컬 실행(정적 템플릿 확인)
1. 루트에서 정적 서버를 실행합니다. (`Pages`/`includes`를 `fetch`로 로드하므로 `file://` 직접 실행은 권장하지 않음)
2. `Pages/index.html`로 접속해 확인합니다.

예시 실행:
```powershell
npx serve .
```

## 배포 (Vercel)

이 레포는 정적 사이트로 Vercel에 배포되도록 설정되어 있습니다. `Pages/` 디렉터리를 사이트 루트로 사용합니다.

간단한 배포 방법:

1. Vercel에 이미 리포를 import 하셨다면, Vercel 대시보드에서 `Deploy` 버튼을 눌러 배포를 시작하세요.
2. 수동으로 배포하려면 로컬에서 Vercel CLI를 사용합니다 (토큰 필요):

```bash
npm i -g vercel
vercel --prod
```

3. 배포 도중 자원 경로가 잘못 보이면 `vercel.json` 또는 파일 경로를 조정하세요. 현재 `vercel.json`이 `Pages/`를 루트로 라우팅합니다.

배포 완료 후 저는 사이트 접속을 확인하고, `Map` 임베드가 정상 동작하는지 콘솔 에러를 검사하겠습니다.
